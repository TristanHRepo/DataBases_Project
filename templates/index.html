Welcome to Plant Parenting!

This web application is an admin-facing application for a database to make basic CRUD (Create, Read, Update, Delete) operations simple.

This database has 5 entities: Users, Plants, Care, Guides, and Experts. Each entity has its own page where you can clearly see (read) all the entries.

You can also insert (create) into each entity table, and edit and delete from each entity table. Intersection tables have their own pages with all CRUD functionality as well.

Users can have multiple Plants, write multiple Guides, and be Experts in multiple fields. The relationship between Users and Plants is nullable.

A type of Plant can be owned by many Users and has one Care requirement associated with it. There is a filtered search on the Plants page.

One Care requirement can be associated with many Plants.

A Guide can be associated with one Plant and one User, but Users can write many Guides and Plants can have many Guides written about them.

Finally, an Expert tag can be held by many Users and Users can have many Expert tags.

We hope you find this app to be a useful tool!
